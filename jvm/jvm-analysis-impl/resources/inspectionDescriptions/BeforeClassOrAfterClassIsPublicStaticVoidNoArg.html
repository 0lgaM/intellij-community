Reports JUnit 4 @BeforeClass and @AfterClass annotated methods and JUnit 5 @BeforeAll and @AfterAll annotated methods that are not public, not static, whose return type is not void or take arguments without declaring a parameter resolver. Such methods are easy to create inadvertently and will not be executed by JUnit tests runners.

Example:


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) { ... }
  }

After the quick-fix is applied:


  class MainTest {
    @BeforeClass
    public static void beforeClass() { ... }
  }
