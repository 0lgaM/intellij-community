Reports FileInputStream or FileOutputStream constructors in cases when it is possible to replace them with Files.newInputStream or Files.newOutputStream accordingly.
Streams that are created using Files methods usually are more efficient than regular ones.

Example:


InputStream is = new BufferedInputStream(new FileInputStream(file));

After quick-fix is applied:


InputStream is = new BufferedInputStream(Files.newInputStream(file.toPath()));

This inspection only reports if the language level of the project or module is 7 or higher.
